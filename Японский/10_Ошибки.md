## 20. Частые ошибки русскоговорящих

### 20.1 Произношение

| Ошибка | Правильно | Пояснение |
|--------|-----------|----------|
| Читать は как «ха» в частицах | [wa] | は как частица = [wa] |
| Читать へ как «хэ» в частицах | [e] | へ как частица = [e] |
| Читать を как «во» | [o] | を = [o] |
| Долгое и краткое не различать | 必сь различать | おばさん vs おばあさん |
| ん читать как «н» всегда | Зависит от контекста | Перед m/b = [m], перед k/g = [ŋ] |
| う в конце слога проглатывать | Произносить, но коротко | です = [desu], не [des] |
| 促音 っ не произносить | Задержка + удвоение следующей | きって = [kitte] |
| Долгие гласные не удлинять | Обязательно удлинять | おばさん ≠ おばあさん |
| り, る, れ, ら,ろ путать | Все = одноударный [ɾ] | Как «р» в слове «оратор» |
| ふ читать как «фу» | Как задувание свечи | Губы не касаются зубов |

### 20.2 Грамматика

| Ошибка | Правильно | Пояснение |
|--------|-----------|----------|
| 私はを食べます | 私は食べます | は не ставится с を |
| 私が行きます (без контекста) | 私は行きます | が = конкретный субъект |
| 食べますを | 食べます (без を) | を = только с глаголом действия |
| 昨日は行きます | 昨日は行きました | Прошедшее время! |
| 日本語は上手です | 私は日本語が上手です | Нужно указать субъект |
| 食べたくないです | 食べたくないです | Правильно (ない = отрицание) |
| 見える / 聞こえる = могу | 見れる / 聞ける = могу | 見える = «видно», 見れる = «могу смотреть» |
| 〜だ + です (冗бдование) | 〜です | НЕЛЬЗЯ: 学生だです |
| 彼は学生だです (冗бдование) | 彼は学生です | Уже с です → нет だ |
| 〜を食べたくないです | 〜が食べたくないです | С たい = が, не を |
| 本を読みます (постоянно) | 本を読みます | Правильно (без контекста = обычное действие) |
| 三時があります | 三時に起きます | に = точка во времени |
| 起きる在床上 (отриц.) | 起きない在床上 | Не ます-форма в обычном отрицании |

### 20.3 Кандзи

| Ошибка | Правильно | Пояснение |
|--------|-----------|----------|
| Путать 大 (большой) и 太 (толстый) | 大 = большой, 太 = толстый | Разные кандзи |
| Путать 未 (ещё не) и 末 (конец) | 未 = «ещё не», 末 = «конец» | Разная форма |
| Путать 日 (день) и 目 (глаз) | 日 = солнце/день, 目 = глаз | Разные формы |
| Путать 土 (земля) and 士 (солдат) | 土 = земля, 士 = воин | Разная верхняя часть |
| Путать 午 (полдень) and 牛 (корова) | 午 = полдень, 牛 = корова | Обратить внимание на наклон |
| Путать 末 (конец) и 未 (ещё не) | 末 = конец (длинная линия), 未 = ещё не (короткая) | |

### 20.4 Лексика и употребление

| Ошибка | Правильно | Пояснение |
|--------|-----------|----------|
| 私は田中を知っています | 私は田中さんを知っています | Нужно さん |
| すみません = извините | すみません = извините / простите / простите | Шире, чем «извините» |
| お願いします = прошу | お願いします = прошу / пожалуйста | Только в просьбах |
| いいです = хорошо | いいです = хорошо / нет, спасибо | Может означать отказ! |
| 大丈夫 = в порядке | 大丈夫 = всё хорошо / можно / нормально | Может означать «да» или «нет» |
| ちょっと... = немного | ちょっと... = немного / нет, спасибо | Часто = вежливый отказ |
| お元気ですか = привет | お元気 как = формальное «как дела?» | Не для друзей |
| はい = да всегда | はい = да / понял | Иногда = «понял», не «согласен» |
| いいえ = нет | いいえ = нет / нет, не так | Может означать «нет, наоборот» |

### 20.5 Вежливость (кейго)

| Ошибка | Правильно | Пояснение |
|--------|-----------|----------|
| Использовать кейго с друзьями | Обычная форма | Кейго = только с чужими/начальниками |
| お食べます (своя еда) | 食べます | お- = не для своих действий |
| おっしゃる (свой глагол) | 言う | おっしゃる = только чужой глагол |
| 行かれる (простая пассивная) | 行かれる = уважаемый | Пассивная форма = возвышающая |
| 存じます (разговорн.) | 知っています | 存じます = формально |
| させていただく (без необходимости) | Простая форма | Часто избыточно |

### 20.15 Таблица спряжения て-формы (подробная)

**I класс (U-глаголы):**

| Окончание | て-форма | Пример | Перевод |
|-----------|---------|--------|---------|
| -う | って | 買う → �買って | Покупать |
| -つ | って | 待つ → 待って | Ждать |
| -る | って | 作る → 作って | Делать |
| -く | いて | 書く → 書いて | Писать |
| -ぐ | いで | 泳ぐ → 泳いで | Плавать |
| -す | して | 話す → 話して | Говорить |
| -ぬ | んで | 死ぬ → 死んで | Умирать |
| -ぶ | んで | 遊ぶ → 遊んで | Играть |
| -む | んで | 読む → 読んで | Читать |

**Исключения I класса:**
| Глагол | Правильно | Неправильно |
|--------|-----------|-------------|

| 帰る | 帰って | *帰いて |
| 切る | 切って | *切いて |
| 知る | 知って | *知いて |
| 入る | 入って | *入いて |
| 走る | 走って | *走いて |

### 20.36 Как исправлять ошибки

**Пошаговый процесс:**

| Шаг | Действие |
|-----|----------|
| 1 | Заметь ошибку |
| 2 | Назови причину |
| 3 | Найди соответствующий материал |
| 4 | Практикуй паттерн заново |
| 5 | Проверь, повторяется ли ошибка |

**Советы:**
1. Веди дневник ошибок
2. Повторяй ошибки через 1 день, 3 дня, 7 дней
3. Учись на чужих ошибках
4. Не бойся ошибаться — ошибки = обучение
5. Практикуйся с реальными тестами
