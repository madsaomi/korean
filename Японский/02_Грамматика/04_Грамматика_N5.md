## 6. Структура предложения и частицы
### 6.1 Базовый порядок слов (SOV)
**Русский (SVO):** Я ем суши.
**Японский (SOV):** 私はすしを食べます。

```
[Тема] は [Объект] を [Глагол]
[Тема] は [Прилагательное] です
[Тема] は [Существительное] です
```
### 6.2 Частицы — полный разбор
#### は (wa) — тема
Указывает, о чём/о ком говорится. Читается [wa], не [ha].

**Основные правила:**
- Тема предложения: о чём идёт речь
- Не используется в подлежащем, если оно уже известно
- Не ставится в косвенных падежах (кроме темы)

| Пример | Перевод | Комментарий |
|--------|---------|------------|
| 私は学生です。 | Я — студент. | Тема: «что касается меня» |
| 日本語は面白いです。 | Японский — интересный. | Тема: «что касается японского» |
| 今日は暑いです。 | Сегодня жарко. | Тема: «что касается сегодняшнего дня» |
| 田中さんは会社員です。 | Танака — офисный работник. | Тема: «что касается Танака» |
| 東京は大きいです。 | Токио — большой город. | Тема: «что касается Токио» |

**Различные типы тем:**
- Личная тема: 私は田中です。 (Я — Танака.)
- Общая тема: 日本語は難しいです。 (Японский — сложный.)
- Контрастная тема: 肉は食べますが、魚は食べません。 (Мясо ем, а рыбу — нет.)
- Тема времени: 今日は日曜日です。 (Сегодня — воскресенье.)

**Важно:** В ответах на вопросы は часто опускается:
- Q: 誰ですか。 A: 田中です。 (Танака.)
- Q: 何ですか。 A: コーヒーです。 (Кофе.)
#### が (ga) — субъект
| Функция | Пример | Комментарий |
|---------|--------|------------|
| Субъект действия | 山田さんが来ました。 | Пришёл именно Ямада |
| Субъект состояния | 本があります。 | Есть книга (констатация) |
| Объект желания | 水が飲みたい。 | Хочу воды |
| Вопросительное | 誰が来ましたか。 | Кто пришёл? |
| Краткий ответ | 田中さんが。 | Ямада (именно он) |

**Разница は vs が (подробно):**

| Ситуация | は | が |
|----------|----|----|
| Тема разговора | 日本語は面白い。 | — |
| Новый объект | — | 友達が来た。 |
| Контраст | 肉は食べるが… | — |
| Уникальность | — | 富士山が高い。 |
| Вопрос с «кто» | — | 誰が来た？ |
| Ограничение | — | 自分がやる。 |

**Примеры контраста:**
- 肉は食べますが、魚は食べません。 (Мясо ем, рыбу — нет.)
- 日本語はできますが、中国語はできません。 (Японский могу, китайский — нет.)
#### を (wo/o) — прямой объект
**Основные правила:**
- Указывает прямой объект действия
- Читается [o], не [wo]
- Опускается в разговорной речи

| Пример | Перевод |
|--------|---------|
| 水を飲みます。 | Пью воду. |
| 本を読む。 | Читаю книгу. |
| コーヒーを飲んでください。 | Выпейте кофе. |
| テレビを見ます。 | Смотрю телевизор. |

**Способы/средства (с で):**
- 電車で行きます。 (Еду на поезде.)
- 自転車で行きます。 (Еду на велосипеде.)

**Движение через (с を):**
- 道を歩きます。 (Иду по дороге.)
- 空を飛びます。 (Лечу по небу.)
- 橋を渡ります。 (Пересекаю мост.)
#### に (ni) — цель/место/время/адресат
**Основные функции:**

| Функция | Пример | Комментарий |
|---------|--------|------------|
| Точка во времени | 九時に起きる | Встаю в 9 |
| Направление | 学校に行く | Иду в школу |
| Адресат | 先生に会う | Встречаю учителя |
| Цель | 貭い物に行く | Иду за покупками |
| Местонахождение | 机の上にある | На столе |
| Превращение | 医者になる | Становлюсь врачом |
| Цель действия | 入口に入る | Вхожу через вход |
| Результат | 三つに分ける | Разделить на три |
| Время начала | 三月から九月まで | С марта по сентябрь |
| Частота | 毎日走る | Каждый день бегаю |

**Примеры:**
- 六時に起きます。 (Встаю в 6.)
- 東京に行きます。 (Еду в Токио.)
- 友達に会います。 (Встречаюсь с другом.)
- 会社に入ります。 (Устраиваюсь на работу.)
- 食べ物を買う。 (Покупаю еду.)
#### で (de) — место/средство
| Функция | Пример | Комментарий |
|---------|--------|------------|
| Место действия | 教室で勉強する | Учусь в классе |
| Средство | 電車で行く | Еду на поезде |
| Язык | 日本語で話す | Говорю по-японски |
| Причина | 病気で休む | Из-за болезни |
| Материал | 木で作る | Сделано из дерева |
| Лимит | 千円で買う | Покупаю за 1000 иен |

**Примеры:**
- レストランで食べます。 (Ем в ресторане.)
- ペンで書ます。 (Пишу ручкой.)
- 英語で話します。 (Говорю по-английски.)
- 風邪で休みます。 (Болею простудой.)
#### へ (e) — направление
Читается [e], не [he]. Отличается от に тем, что подчёркивает **направление**, а не точку назначения.

| Пример | Комментарий |
|--------|------------|
| 東京へ行く | Еду в Токио (направление) |
| 学校へ行ってください | Идите в школу |
| 日本へ来ました | Приехал в Японию |
#### と (to) — совместность/цитата/условие
| Функция | Пример | Комментарий |
|---------|--------|------------|
| И (перечисление) | これとそれ | Это и то |
| С (совместность) | 友達と行く | Иду с другом |
| Цитата | 行くと言った | Сказал, что пойдёт |
| Условие | 雨だと行かない | Если дождь — не пойду |
| Выбор (в вопросе) | コーヒーとお茶、どちらがいい | Кофе или чай? |

**Примеры:**
- 私と友達で行きます。 (Иду с другом.)
- 「はい」と言いました。 (Сказал «да».)
#### の (no) — принадлежность
| Функция | Пример | Комментарий |
|---------|--------|------------|
| Притяжательность | 私の本 | Моя книга |
| Связь | 日本語の先生 | Учитель японского |
| Номинализация | 読むのが好き | Люблю читать |
| Уточнение | 東京の大学 | Университет в Токио |
| Причина | 雨のため | Из-за дождя |

**Примеры:**
- 私の名前は田中です。 (Моё имя — Танака.)
- 日本語の本を買いました。 (Купил книгу по-японски.)
- 食べるのが好きです。 (Люблю есть.)
#### も (mo) — тоже/даже
| Функция | Пример | Комментарий |
|---------|--------|------------|
| Тоже | 私も行く | Я тоже пойду |
| Даже | 子供もできる | Даже ребёнок может |
| Ни (+отриц.) | 水もない | Нет даже воды |
| Все/каждый | みんなも来た | Все тоже пришли |
| Также | これも美味しい | Это тоже вкусно |

**Примеры:**
- 私も日本語を勉強しています。 (Я тоже учу японский.)
- 水も食べ物もない。 (Нет ни воды, ни еды.)
#### から (kara) — от/после/потому что
| Функция | Пример | Комментарий |
|---------|--------|------------|
| Откуда | 東京から来た | Из Токио |
| С (время начала) | 九時から始まる | С 9 часов |
| Причина | 雨だから行かない | Из-за дождя не пойду |
| Потому что (разговорн.) | 寒いから窓を閉める | Холодно, закрою окно |
| С каких пор | 先月から | С прошлого месяца |

**Примеры:**
- 駅から五分です。 (5 минут от вокзала.)
- 三時から五時まで。 (С 3 до 5.)
- 遅いから帰ります。 (Поздно, иду домой.)
#### まで (made) — до
| Пример | Комментарий |
|--------|------------|
| 駅まで歩く | Иду до вокзала |
| 五時まで待つ | Жду до 5 |
| いつまでですか。 | До какого времени? |
| 子供まで | Даже дети |
#### より (yori) — сравнение
| Пример | Комментарий |
|--------|------------|
| 猫より犬が好き | Люблю собак больше кошек |
| 東京は大阪より大きい | Токио больше Осаки |
| このほうがいい | Этот лучше |
#### か (ka) — вопрос
| Функция | Пример | Комментарий |
|---------|--------|------------|
| Вопрос (да/нет) | 行きますか。 | Идёте? |
| Альтернатива | コーヒーかお茶 | Кофе или чай |
| Не знаю (кто/что) | 誰か知っていますか。 | Кого-нибудь знаете? |
| Каждый | 週末何かします | Каждые выходные делаю что-то |

**Примеры:**
- これは何ですか。 (Что это?)
- 田中さんですか。 (Вы Танака?)
#### よ (yo) — уверение
| Пример | Комментарий |
|--------|------------|
| 大丈夫ですよ。 | Всё в порядке! (убеждаю) |
| これ、おいしいよ。 | Это вкусно! (сообщаю) |
| 明日は雨ですよ。 | Завтра будет дождь! |
#### ね (ne) — согласие/подтверждение
| Пример | Комментарий |
|--------|------------|
| いい天気ですね。 | Хорошая погода, да? |
| そうですか。 | Понятно. |
| 行きますね。 | Идём, да? |

**Использование:**
- В конце предложения для подтверждения: そうですね。
- Для выражения удивления: そうですか。
- Для выражения сомнения: そうですかね。
### 6.3 です — связка «быть»
| Форма | Пример |
|-------|--------|
| Наст. утверд. | 学生です。 |
| Наст. отриц. | 学生じゃないです / 学生ではありません。 |
| Прош. утверд. | 学生でした。 |
| Прош. отриц. | 学生じゃなかったです。 |

## 7. Глаголы: классы и основы
### 7.1 Три класса глаголов
| Класс | Окончание | Примеры |
|-------|-----------|---------|
| **I (Годан / U-глаголы)** | -う, -く, -ぐ, -す, -つ, -ぬ, -ぶ, -む, -る | 書く, 読む, 話す |
| **II (Итидан / Ru-глаголы)** | -る (1 слог перед ним) | 食べる, 見る, 起きる |
| **III (Неправильные)** | する, 来る | 勉強する, 来る |

**Исключения (выглядят как II, но класс I):** 切る, 走る, 入る, 帰る, 知る, 要る
### 7.2 Таблица спряжения
| Форма | I (書く) | II (食べる) | III (する) | III (来る) |
|-------|---------|-----------|-----------|-----------|
| 辞書形 (словарная) | 書く | 食べる | する | 来る |
| ます形 (вежл.) | 書きます | 食べます | します | 来ます |
| ない形 (отриц.) | 書かない | 食べない | しない | 来ない |
| て形 | 書いて | 食べて | して | 来て |
| た形 (прош.) | 書いた | 食べた | した | 来た |
| ば形 (условн.) | 書けば | 食べれば | すれば | 来れば |
| 意向形 (давай) | 書こう | 食べよう | しよう | 来よう |
| 可能形 (мочь) | 書ける | 食べられる | できる | 来られる |
| 受身形 (пассив) | 書かれる | 食べられる | される | 来られる |
| 使役形 (каузатив) | 書かせる | 食べさせる | させる | 来させる |

## 8. Формы глаголов N5
### 8.1 ます/ません/ました/ませんでした
| Группа | ます | ません | ました | ませんでした |
|-------|------|--------|--------|------------|
| I: 書く | 書きます | 書きません | 書きました | 書きませんでした |
| II: 食べる | 食べます | 食べません | 食べました | 食べませんでした |
| III: する | します | しません | しました | しませんでした |
| III: 来る | 来ます(きます) | 来ません | 来ました | 来ませんでした |

**Пример:** 毎日勉強します。 (Каждый день учусь.)
### 8.2 Plain (словарная форма)
| Тип | Утверд. | Отриц. | Прош.утв. | Прош.отр. |
|-----|---------|--------|-----------|-----------|
| I | 書く | 書かない | 書いた | 書かなかった |
| II | 食べる | 食べない | 食べた | 食べなかった |
| する | する | しない | した | しなかった |
| 来る | 来る | 来ない | 来た | 来なかった |
### 8.3 て-форма — важнейшая!
**Образование:**

| I класс | Правило | Пример |
|---------|---------|--------|
| -う, -つ, -る | って | 買う→買って, 待つ→待って, 作る→作って |
| -く | いて | 書く→書いて |
| -ぐ | いで | 泳ぐ→泳いで |
| -す | して | 話す→話して |
| -ぬ, -ぶ, -む | んで | 死ぬ→死んで, 遊ぶ→遊んで, 読む→読んで |

**II класс:** -る→て (食べる→食べて) | **III:** する→して, 来る→来て

**Исключение:** 行く→行って (не *行いて)

**Использование:**

| Конструкция | Пример | Перевод |
|------------|--------|---------|
| Связь | 起きて、食べて、行く。 | Встаю, ем, иду. |
| Просьба (てください) | 待ってください。 | Подождите. |
| Прогресс (ている) | 本を読んでいる。 | Читаю книгу. |
| Разрешение (てもいい) | 座ってもいい？ | Можно сесть? |
| Запрет (てはいけない) | 座ってはいけない。 | Нельзя садиться. |
### 8.4 た-форма (прошедшая plain)
Как て-форма, но て→た, で→だ: 買った, 書いた, 読んだ, 食べた, した, 来た.
### 8.5 ない-форма (отрицание)
**I класс:** основа -a + ない (書かない, 読まない, 話さない)
**II класс:** убрать -る + ない (食べない)
**III:** しない, 来ない(こない)
### 8.6 〜ている (процесс / состояние)
| Значение | Пример |
|----------|--------|
| Процесс | 本を読んでいる。 (Читаю книгу.) |
| Состояние | 結婚している。 (Женат.) |
| Привычка | 毎日歩いている。 (Хожу каждый день.) |
| Результат | ドアが開いている。 (Дверь открыта.) |

**Разговорно:** 〜てる (読んでる)
### 8.7 〜たことがある (опыт)
**Пример:** 日本に行ったことがある。 (Был в Японии.) | 食べたことがない。 (Никогда не ел.)
### 8.8 〜つもり (намерение)
`辞書形/ない + つもりだ`: 大学に行くつもりだ。 (Намерен пойти в универ.)
### 8.9 〜たい (хотение)
`ます-основа + たい` — ведёт себя как い-прилагательное

| Спряжение | Пример |
|-----------|--------|
| たい | 行きたい |
| たくない | 行きたくない |
| たかった | 行きたかった |
### 8.10 〜ながら (одновременность)
`ます-основа + ながら`

**Значение:** Делать два действия одновременно.

| Пример | Перевод | Комментарий |
|--------|---------|------------|
| 音楽を聞きながら勉強する。 | Учусь, слушая музыку. | |
| 歩きながら話す。 | Разговариваю на ходу. | |
| 食べながらテレビを見る。 | Смотрю телевизор, пока ем. | |
| 知りながら言わない。 | Знаю, но не говорю. | Уступительное |

**Важно:**
- Используется только с действиями, которые можно делать одновременно
- НЕ используется: 死にながら… (умирать + делать что-то)

**Уступительное значение:**
- 知らないふりをしながら… (притворяюсь, что не знаю…)
- 悪いと知りながら… (знать, что плохо, но…)
- いけないと知りながら… (знать, что нельзя, но…)
### 8.11 〜前 / 〜後 / 〜時
| Конструкция | Форма | Пример | Перевод |
|-------------|-------|--------|---------|
| する前に | ДО действия | 食べる前に手を洗う | Перед едой мою руки |
| した後で | ПОСЛЕ действия | 食べた後で歯を磨ить | После еды чищу зубы |
| する時 | КОГДА | 行く時、本を読む | Когда иду — читаю книгу |
| している時 | КОГДА (процесс) | 食べている時、電話が鳴った | Когда ел — позвонили |

**Примеры:**
- 学校に行く前に、ご飯を食べます。 (Перед школой ем.)
- 勉強した後で、テレビを見ます。 (После учёбы смотрю телевизор.)
- 日本に来た時、何も知りませんでした。 (Когда приехал в Японию, ничего не знал.)
### 8.12 〜たびに (каждый раз когда)
| Пример | Перевод |
|--------|---------|
| 日本に行くたびに、新しい所を見る。 | Каждый раз когда еду в Японию, вижу новые места. |
| 会うたびに、きれいになる。 | Каждый раз когда встречаю — становится красивее. |
### 8.13 〜ところ (место / момент)
| Пример | Перевод |
|--------|---------|
| 勉強しているところです。 | Как раз сейчас занимаюсь. |
| 食べたところです。 | Только что поел. |
| 行くところです。 | Вот-вот пойду. |
### 8.14 〜ようとする (собираться / пытаться)
| Пример | Перевод |
|--------|---------|
| 出かけようとした時、雨が降った。 | Когда собрался выйти — пошёл дождь. |
| 開けようとしたが、開かなかった。 | Пытался открыть, но не открылось. |
### 8.15 〜ことがある (иногда)
| Пример | Перевод |
|--------|---------|
| 時遅刻することがあります。 | Иногда опаздываю. |
| 天気が悪いことがあります。 | Иногда погода плохая. |
### 8.16 〜はずがない (не может быть)
| Пример | Перевод |
|--------|---------|
| そんなはずがない。 | Такого не может быть. |
| 彼が来るはずがない。 | Он не может прийти. |
### 8.17 〜ことはない (нет необходимости)
| Пример | Перевод |
|--------|---------|
| 心配することはない。 | Некогда беспокоиться. |
| 急ぐことはない。 | Некуда спешить. |
### 8.18 〜ないでください (не делайте)
| Пример | Перевод |
|--------|---------|
| 運転しないでください。 | Не водите машину. |
| 写真を撮らないでください。 | Не фотографируйте. |
### 8.19 〜までに (до момента)
| Пример | Перевод |
|--------|---------|
| 三時に来てください。 | Приходите к 3 часам. |
| 金曜日までに提出してください。 | Представьте до пятницы. |
### 8.20 〜ようにする / 〜ようになる
| Конструкция | Пример | Перевод |
|-------------|--------|---------|
| ようにする | 毎日勉強するようにしている | Стараюсь учиться каждый день |
| ようになる | 日本語が話せるようになった | Стал мочь говорить по-японски |
| ようにする | 遅刻しないようにする | Стараюсь не опаздывать |
| ようになる | 三時に起きるようになった | Привык вставать в 3 |
### 8.21 〜ことにする / 〜ことになる
| Конструкция | Пример | Перевод |
|-------------|--------|---------|
| ことにする | 来年、日本に行くことにした | Решил поехать в Японию в след. году |
| ことになる | 来年、日本に行くことになった | Мне сказали ехать в Японию |
| しないことにした | 行かないことにした | Решил не ехать |
| ことになった | 留学することになった | Решено — еду на стажировку |

**Разница:** ことにする — личное решение; ことになる — внешнее решение/распоряжение.
### 8.22 〜そう (похоже на то что)
| Тип | Форма | Пример | Перевод |
|-----|-------|--------|---------|
| Нал./прил. (по виду) | 語幹+そう | 美味しそう | Выглядит вкусно |
| Глагол (по виду) | ます-осн.+そう | 雨が降りそう | Похоже, будет дождь |
| Отриц. (похоже не) | なさそう | 来なさそう | Похоже, не придёт |
| Слух (говорят) | Plain+そうだ | 明日は雨だそうだ | Говорят, завтра дождь |

**Важно:** Если корень на い → убрать い: 美味しい → 美味しそう. Если корень на な → добавить そう: 静か → 静かそう.
### 8.23 〜ようだ / 〜みたいだ (похоже на то что)
| Конструкция | Пример | Перевод |
|-------------|--------|---------|
| ようだ (письм.) | 猫のようだ | Похож на кошку |
| みたいだ (разг.) | 猫みたいだ | Похож на кошку |
| ような + сущ. | 猫のような動物 | Животное, похожее на кошку |
| ように + глагол | 猫のように歩く | Идти, как кошка |
| みたいな + сущ. | 猫みたいな人 | Человек, похожий на кошку |
| みたいに + глагол | 猫みたいに走ает | Бегать, как кошка |

**Примеры:**
- 風邪のようだ。 (Похоже на простуду.)
- 日本語が話せるようになりたい。 (Хочу научиться говорить по-японски.)
- 彼は先生みたいだ。 (Он похож на учителя.)
### 8.24 〜という (называется / говорят что)
| Функция | Пример | Перевод |
|---------|--------|---------|
| Название | これは何という花ですか。 | Как называется этот цветок? |
| Имя | 田中という人 | Человек по имени Танака |
| Слух | 明日は雨だという | Говорят, завтра дождь |
| Определение | 「敬語」とは何か。 | Что такое «кейго»? |
### 8.25 〜ことがある (бывает что / иногда)
| Функция | Пример | Перевод |
|---------|--------|---------|

| Опыт (было ли) | 日本に行ったことがある。 | Был в Японии. |
| Не бывает (отр.) | 日本に行ったことがない。 | Никогда не был в Японии. |
### 8.26 〜はずだ (должно быть / ожидается)

| Прошлое | 来るはずだった。 | Должен был прийти. |

### 8.27 〜ことはない (нет необходимости)
| Пример | Перевод |

### 9.12 Сравнение: подробные примеры
**Базовые конструкции:**

| Структура | Пример | Перевод |
|-----------|--------|---------|
| AはBより～ | 東京は大阪より大きい | Токио больше Осаки |
| BよりAのほうが～ | 大阪より東京のほうが大きい | Токио больше Осаки |
| AとBとどちらが～ | 東京と大阪とどちらが大きい | Токио или Осака — что больше? |
| 一番～ | 日本で一番高い山 | Самая высокая гора в Японии |
| ～ほど～ない | 大阪は東京ほど高くない | Осака не так дорога, как Токио |

**Примеры с прилагательными:**

| Сравнение | Пример |
|-----------|--------|

| 新しい vs 古い | 新しい靴を買いました。古い靴を捨てました。 |
| 大きい vs 小さい | 大きい犬が好きです。小さい猫も好きです。 |
| 楽しい vs 面白い | 映画は楽しいです。この本は面白いです。 |

## 12. Структура предложения — подробные примеры
### 12.1 Базовые модели
**Модель 1: SはOです**
```
私は      水を      飲みます。
Watashi wa mizu wo nomimasu.
Я        воду     пью.
```

**Модель 2: SはAです**
```
私は      学生です。
Watashi wa gakusei desu.
Я         студент.
```

**Модель 3: SはV**
```
猫が      走っています。
Neko ga   hashitte imasu.
Кошка     бежит.
```
### 12.2 Полные примеры с контекстом
**Диалог в магазине:**
> **A:** すみません、この本はいくらですか。
> **B:** 五百円です。
> **A:** じゃあ、これを二冊ください。
> **B:** かしこまりました。一千円になります。

**Диалог в ресторане:**
> **A:** ご注文はお決まりですか。
> **B:** ラーメンと餃子をお願いします。
> **A:** ラーメンにしますか、つけ麺にしますか。
> **B:** ラーメンでお願いします。

**Описание места:**
> 東京は大きい都市です。人口は一千万人以上です。新しいビルがたくさんあります。交通も便利です。地下鉄や電車でどこへでも行けます。
### 12.3 Порядок слов в предложении
**Базовый порядок:** [Подлежащее] は [Объект] を [Место] で [Глагол]

| Элемент | Пример | Функция |
|---------|--------|---------|
| 主語 (подлежащее) | 私は | Тема/субъект |
| 目的語 (объект) | 本を | Прямой объект |
| 場所 (место) | 図書館で | Место действия |
| 時間 (время) | 毎日 | Время |
| 方法 (способ) | 電車で | Средство |
| 述語 (сказуемое) | 読みます | Глагол |

**Пример:**
> 私は毎日図書館で本を読みます。
> (Я каждый день читаю книги в библиотеке.)
### 12.4 Усложнение предложения
**Добавление деталей:**
> 私は毎朝七時に起きて、ご飯を食べて、学校に行きます。
> (Я каждый день утром встаю в 7, ем завтрак и иду в школу.)

**Связывание предложений:**
> 日本語を勉強しています。毎日三時間勉強します。
> (Учу японский. Каждый день учусь 3 часа.)

**Выражение мнения:**
> 私は日本語は面白いと思います。
> (Я думаю, что японский — интересный.)

### 12.9 Полная таблица て-формы
**Правило 1: Последняя буква う-ряд -gt って**
- かう → 買って (покупать)
- まつ → 待って (ждать)
- つくる → 作って (делать)

**Правило 2: く -gt いて, ぐ -gt いで**
- かく → 書いて (писать)
- およぐ → 泳いで (плавать)

**Правило 3: す -gt して**
- はなす → 話して (говорить)

**Правило 4: ぬ, ぶ, む -gt んで**
- しぬ → 死んで (умирать)
- あそぶ → 遊んで (играть)
- よむ → 読んで (читать)

**Правило 5: II класс - просто て**
- たべる → 食べて (есть)
- みる → 見て (смотреть)

**Правило 6: III класс**
- する → して (делать)
- くる → 来て (приходить)
### 12.10 Полная таблица ない-формы
**I класс: основа-a + ない**
- かく → 書かない (не писать)

- はなす → 話さない (не говорить)
- かう → 買わない (не покупать)

## 13. Подробные объяснения грамматики N5
### 13.1 Частица は (wa) — подробно
**Функция:** Указывает тему предложения (о чём говорится).

**Правила:**
- Читается [wa], не [ha]
- Ставится после темы
- Не ставится в подлежащем, если оно уже известно

**Примеры:**

| Предложение | Перевод | Комментарий |
|-------------|---------|------------|
| 私は学生です | Я — студент | Тема: «что касается меня» |
| 日本語は面白い | Японский интересный | Тема: «что касается японского» |
| 今日は暑い | Сегодня жарко | Тема: «что касается сегодня» |
| 田中さんは会社員です | Танака — офисный работник | Тема: «что касается Танака» |
| コーヒーは飲みません | Кофе не пью | Контраст: «а что касается кофе — не пью» |

**Когда НЕ использовать は:**
- В косвенных падежах (кроме темы)
- В подлежащем, если оно уже известно
- В вопросах с «кто» (тогда используется が)
### 13.2 Частица が (ga) — подробно
**Функция:** Указывает субъект действия или констатирует факт.

**Правила:**
- Ставится после субъекта
- Используется в вопросах с «кто» и «что»
- Используется с глаголами существования (ある/いる)
- Используется с прилагательными (好き, 上手, etc.)

**Примеры:**

| Предложение | Перевод | Комментарий |
|-------------|---------|------------|
| 猫がいます | Есть кошка | Субъект существования |
| 本があります | Есть книга | Субъект существования |
| 誰が来ましたか | Кто пришёл? | Вопрос с «кто» |
| 何が好きですか | Что нравится? | Вопрос с «что» |
| 魚が好きです | Люблю рыбу | Прилагательное (好き) |
| 日本語ができます | Могу (говорить) по-японски | Прилагательное (できる) |

**Разница は vs が:**

| Ситуация | は | が |
|----------|----|----|
| Тема разговора | 日本語は面白い | — |
| Новый объект | — | 友達が来た |
| Контраст | 肉は食べるが… | — |
| Уникальность | — | 富士山が高い |
| Вопрос с «кто» | — | 誰が来た? |
| Ограничение | — | 自分がやる |
### 13.3 Частица を (wo) — подробно
**Функция:** Указывает прямой объект действия.

**Правила:**
- Ставится после прямого объекта
- Читается [o], не [wo]
- Опускается в разговорной речи
- Может использоваться для обозначения движения через (道を歩く)

**Примеры:**

| Предложение | Перевод | Комментарий |
|-------------|---------|------------|
| 水を飲みます | Пью воду | Прямой объект |
| 本を読む | Читаю книгу | Прямой объект |
| 道を歩きます | Иду по дороге | Движение через |
| 空を飛びます | Лечу по небу | Движение через |
| 橋を渡ります | Пересекаю мост | Движение через |
### 13.4 Частица に (ni) — подробно
**Функция:** Указывает направление, время, адресат, цель.

**Правила:**
- Ставится после направления/времени/адресата
- Используется с глаголами движения (行く, 来る)
- Используется для обозначения точного времени
- Используется для обозначения адресата действия

**Примеры:**

| Предложение | Перевод | Комментарий |
|-------------|---------|------------|
| 学校に行きます | Иду в школу | Направление |
| 六時に起きます | Встаю в 6 | Точное время |
| 先生に会う | Встречаю учителя | Адресат |
| 買い物に行く | Иду за покупками | Цель |
| 医者になる | Становлюсь врачом | Превращение |
### 13.5 Частица で (de) — подробно
**Функция:** Указывает место действия, средство, причину.

**Правила:**
- Ставится после места действия или средства
- Используется с глаголами действий (не существования)
- Используется для обозначения средства передвижения

## 17. て-форма — подробные правила
### 17.1 Правила образования て-формы
**I класс (U-глаголы):**

| Последняя буква | Правило | Пример | て-форма |
|-----------------|---------|--------|---------|
| う, つ, る | って | 買う, 待つ, 作る | 買って, 待って, 作って |
| く | いて | 書く | 書いて |
| ぐ | いで | 泳ぐ | 泳いで |
| す | して | 話す | 話して |
| ぬ, ぶ, む | んで | 死ぬ, 遊ぶ, 読む | 死んで, 遊んで, 読んで |

**Исключения I класса:**

| Глагол | Правильно | Неправильно |
|--------|-----------|-------------|
| 行く | 行って | *行いて |
| 帰る | 帰って | *帰いて |
| 切る | 切って | *切いて |
| 知る | 知って | *知いて |
| 入る | 入って | *入いて |
| 走る | 走って | *走いて |

**II класс (Ru-глаголы):** просто убрать -る и добавить て
- 食べる → 食べて
- 見る → 見て

**III класс (неправильные):**
- する → して
- 来る → 来て (きて)
### 17.2 Использование て-формы
| Конструкция | Значение | Пример |
|-------------|---------|--------|
| て-форма + ください | Просьба | 食べてください |
| ている | Процесс/состояние | 食べている |
| てもいい | Разрешение | 食べてもいいですか |
| てはいけない | Запрет | 食べてはいけない |
| てみる | Попробовать | 食べてみる |
| てしまう | Завершённость | 食べてしまった |
| ておく | Заранее | 食べておく |
| てあげる | Сделать для кого-то | 手伝ってあげる |
| してくれる | Сделать для меня | 手伝ってくれる |
| てもらう | Попросить сделать | 手伝ってもらう |
