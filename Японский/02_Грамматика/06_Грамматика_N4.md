## 12. Грамматика N4 — сложные конструкции
### 12.1 〜てみる (попробовать сделать)
`て-форма + みる`

| Пример | Перевод |
|--------|---------|
| 食べてみる。 | Попробую съесть. |
| 着てみる。 | Примерю (одежду). |
| 行ってみたい。 | Хочу попробовать сходить. |
### 12.2 〜てしまう (завершённость / сожаление)
`て-форма + しまう`

| Функция | Пример | Перевод |
|---------|--------|---------|
| Завершённость | 読んでしまった。 | Прочитал полностью. |
| Сожаление | 忘れてしまった。 | Забыл (к сожалению). |
| Разг. | 食べちゃう / 飲んじゃう | Съесть/выпить (нечаянно) |

**Разговорные сокращения:** 〜てしまう → 〜ちゃう, 〜でしまう → 〜じゃう
- 食べてしまう → 食べちゃう
- 読んでしまう → 読んじゃう
### 12.3 〜ておく (заранее сделать)
`て-форма + おく`

| Пример | Перевод |
|--------|---------|
| 予約しておく。 | Забронировать заранее. |
| 覚えておく。 | Запомнить (на будущее). |
| 洗っておく。 | Помыть заранее. |

**Разговорно:** 〜とく (買っておく→買っとく, 読んでおく→読んどく)
### 12.4 〜てもいい / 〜てはいけない / 〜なくてもいい
| Конструкция | Значение | Пример |
|-------------|----------|--------|
| てもいい | Можно | 座ってもいいですか。 |
| てはいけない | Нельзя | 走ってはいけません。 |
| なくてもいい | Можно не | 待たなくてもいいよ。 |
### 12.5 〜なければならない / 〜なきゃ (долженствование)
| Форма | Значение | Пример |
|-------|----------|--------|
| なければならない | Должен | 勉強しなければならない。 |
| なければいけない | Должен | 行かなければいけない。 |
| なくちゃ (разг.) | Должен | 行かなくちゃ。 |
| なきゃ (разг.) | Должен | 勉強しなきゃ。 |
### 12.6 〜かもしれない (возможно)
`Plain + かもしれない`

| Пример | Перевод |
|--------|---------|
| 雨が降るかもしれない。 | Возможно, будет дождь. |
| 遅れるかもしれない。 | Возможно, опоздаю. |
| 安いかもしれない。 | Может быть, дёшево. |
### 12.7 〜でしょう / 〜だろう (вероятно)
| Пример | Перевод |
|--------|---------|
| 明日は雨でしょう。 | Завтра, наверное, будет дождь. |
| 大丈夫だろう。 | Наверное, всё в порядке. |
### 12.8 〜はず (должно быть / ожидается)
| Пример | Перевод |
|--------|---------|
| 知っているはずだ。 | Должен знать. |
| 来るはずだった。 | Должен был прийти. |
### 12.9 〜つもり (намерение / мнение о себе)
| Пример | Перевод |
|--------|---------|
| 大学に行くつもりだ。 | Намерен пойти в университет. |
| 買うつもりはない。 | Не намерен покупать. |
| 勉強したつもりだ。 | Считаю, что поучился (но мало). |
### 12.10 〜ことにする / 〜ことに決める (решить)
| Форма | Пример | Перевод |
|-------|--------|---------|
| することにした | Решил сделать | 結婚することにした。 |
| しないことにした | Решил не делать | 行かないことにした。 |
### 12.11 〜ようにする (стараться делать)
| Пример | Перевод |
|--------|---------|
| 毎日運動するようにしている。 | Стараюсь заниматься каждый день. |
| 遅刻しないようにする。 | Стараюсь не опаздывать. |
### 12.12 〜ようになる (стать так, что)
| Пример | Перевод |
|--------|---------|
| 日本語が話せるようになった。 | Стал мочь говорить по-японски. |
| 食べられるようになった。 | Стал мочь есть (раньше не мог). |
### 12.13 〜たほうがいい (лучше сделать)
| Пример | Перевод |
|--------|---------|
| 病院に行ったほうがいい。 | Лучше сходить в больницу. |
| 行かないほうがいい。 | Лучше не ходить. |
### 12.14 〜なくちゃ / 〜なきゃ (обязательство, разг.)
| Форма | Пример |
|-------|--------|
| なくちゃ | 行かなくちゃ。 |
| なきゃ | 勉強しなきゃ。 |
### 12.15 〜ながら (несмотря на)
| Пример | Перевод |
|--------|---------|
| 知っていながら言わない。 | Зная, не говорит. |
| 悪いと知りながらやった。 | Сделал, зная что плохо. |
### 12.16 〜のに (хотя / несмотря на)
| Функция | Пример | Перевод |
|---------|--------|---------|
| Уступительная | 勉強したのに合格しなかった。 | Хотя учился, не сдал. |
| Сожаление | 早く来ればよかったのに。 | Хорошо бы прийти пораньше. |

**Важно:** 〜のに звучит сожаление/удивление.
### 12.17 〜ば / 〜なら / 〜たら / 〜と (условия)
| Конструкция | Значение | Пример |
|-------------|----------|--------|
| 〜ば (если) | Универсальное | 安ければ買う。 |
| 〜なら (если, раз) | Контекст | 行くなら、買ってきて。 |
| 〜たら (когда/если) | После события | 着いたら電話する。 |
| 〜と (когда/если) | Естественный результат | 春になると暖かい。 |
### 12.18 〜ても (даже если)
| Пример | Перевод |
|--------|---------|
| 雨でも行く。 | Даже если дождь, пойду. |
| 高くても買う。 | Даже если дорого, куплю. |
### 12.19 〜しか〜ない (только)
| Пример | Перевод |
|--------|---------|
| 水しか飲まない。 | Пью только воду. |
| 百円しかない。 | Есть только 100 иен. |
### 12.20 〜だけ (только, без отрицания)
| Пример | Перевод |
|--------|---------|
| 水だけ飲む。 | Пью только воду. |
| これだけある。 | Есть только это. |
### 12.21 〜ばかり (только / только что)
| Функция | Пример | Перевод |
|---------|--------|---------|
| Только что | 食べたばかりだ。 | Только что поел. |
| Только (повтор) | ゲームばかりしている。 | Только и делает, что играет. |
### 12.22 〜すぎる (слишком)
| Тип | Форма | Пример |
|-----|-------|--------|
| Глагол | осн.+すぎる | 食べ過ぎる (переесть) |
| い-прил. | 語幹+すぎる | 大きすぎる (слишком большой) |
| な-прил. | 語幹+すぎる | 静かすぎる (слишком тихий) |
### 12.23 〜やすい / 〜にくい (легко/трудно)
| Форма | Пример |
|-------|--------|
| ます-осн.+やすい | 書きやすい (легко пишется) |
| ます-осн.+にくい | 読みにくい (трудно читается) |
### 12.24 〜方 (способ действия)
`ます-основа + 方 (かた)`

| Пример | Перевод |
|--------|---------|
| 書き方 | способ письма |
| 食べ方 | способ еды |
| 使い方 | способ использования |
### 12.25 〜ながら (одновременность)
| Пример | Перевод |
|--------|---------|
| テレビを見ながら食べる。 | Ем, смотря телевизор. |
| 歩きながら話す。 | Разговариваю на ходу. |
### 12.26 〜ため (для / из-за)
| Функция | Пример |
|---------|--------|
| Для | 健康のために運動する。 |
| Из-за | 雨のため中止。 |
### 12.27 〜し (перечисление причин)
| Пример | Перевод |
|--------|---------|
| 安いし、美味しいし。 | И дёшево, и вкусно. |
| 頭も痛いし、熱もあるし。 | И голова болит, и температура. |
### 12.28 〜そうだ (по виду / говорят)
| Функция | Пример |
|---------|--------|
| По виду | 美味しそう (выглядит вкусно) |
| Говорят | 天気予報によると、雨だそうだ |
| По виду (гл.) | 雨が降りそう (похоже на дождь) |
### 12.29 〜ようだ / 〜みたいだ (похоже)
| Пример | Перевод |
|--------|---------|
| 猫みたいだ。 | Похож на кошку. |
| 風邪のようだ。 | Похоже на простуду. |
### 12.30 〜という (называется/говорят что)
| Пример | Перевод |
|--------|---------|
| これは何という花ですか。 | Как называется этот цветок? |
| 田中という人。 | Человек по имени Танака. |

## 13. Глаголы действия и состояния
### 13.1 あげる / くれる / もらう (получение-дарение)
| Глагол | Направление | Пример |
|--------|------------|--------|
| あげる | Я → Другой | 友達に本をあげた。 |
| くれる | Другой → Я | 友達が本をくれた。 |
| もらう | Я ← Другой | 友達に本をもらった。 |

**て-форма + auxiliary:**

| Конструкция | Пример |
|-------------|--------|
| てあげる | 手伝ってあげる (помогаю тебе) |
| てくれる | 手伝ってくれる (помогаешь мне) |
| てもらう | 手伝ってもらう (прошу помочь) |

**Вежливые формы:** さしあげる / くださる / いただく
### 13.2 ある / いる (есть/имеется)
| Глагол | Для чего | Пример |
|--------|---------|--------|
| ある | Неодушевлённые | 本がある。 |
| いる | Одушевлённые | 猫がいる。 |

**Местонахождение:** 机の上に本がある。 (Книга на столе.)
### 13.3 できる (мочь / появляться)
| Функция | Пример |
|---------|--------|
| Умение | 日本語ができる。 |
| Появление | 新しい店ができた。 |
| Возможность | できますか？ |

## 15. Пассивный и каузативный залоги
### 15.1 受身形 (пассив)
| Класс | Форма | Пример |
|-------|-------|--------|
| I | a-ряд+れる | 書かれる (написан) |
| II | られる | 食べられる (съеден) |
| III | される/来られる | される, 来られる |

| Пример | Перевод |
|--------|---------|
| 先生に褒められた。 | Был похвален учителем. |
| 犬に噛まれた。 | Был укушен собакой. |
### 15.2 使役形 (каузатив)
| Класс | Форма | Пример |
|-------|-------|--------|
| I | a-ряд+せる | 書かせる (заставить писать) |
| II | させる | 食べさせる (заставить есть) |
| III | させる/来させる | させる, 来させる |

| Пример | Перевод |
|--------|---------|
| 子供に宿題をさせる。 | Заставляю ребёнка делать дз. |
| 先生が学生に読ませる。 | Учитель заставляет читать студента. |
### 15.3 使役受身形 (каузативный пассив)
| Пример | Перевод |
|--------|---------|
| 宿題をさせられた。 | Меня заставили сделать дз. |
| 走らせられた。 | Меня заставили бежать. |

## 16. Вежливость: Кейго (敬語)
### 16.1 Три типа
| Тип | Назначение | Пример |
|-----|-----------|--------|
| 尊敬語 (сонкэйго) | Возвышает субъекта | お書きになる |
| 謙譲語 (кэндзёго) | Принижает себя | お書きする |
| 丁寧語 (тэйнего) | Вежливость вообще | です/ます |
### 16.2 尊敬語 (когда уважаемый — субъект)
**Правило:** お + ます-основа + になる

| Слово | 尊敬語 |
|-------|--------|
| 行く | おいでになる / 行かれる |
| 食べる | 召し上がる (особое) |
| 言う | おっしゃる (особое) |
| する | なさる (особое) |
| 来る | おいでになる |
| 見る | ご覧になる |
| 知っている | ご存じだ |
| くれる | くださる |

**Спряжение особых:** なさる→なさいます, くださる→くださいます, おっしゃる→おっしゃいます
### 16.3 謙譲語 (когда уважаемый — объект)
**Правило:** お + ます-основа + する / ご + 漢語 + する

| Слово | 謙譲語 |
|-------|--------|
| 行く/来る | 参る (особое) |
| 言う | 申す (особое) |
| する | 致す (особое) |
| 食べる | いただく |
| 見る | 拝見する |
| 知っている | 存じている |
| もらう | いただく |
| あげる | さしあげる |
| 聞く | 伺う |
| 会う | お目にかかる |
### 16.4 Образование (стандартное)
| Форма | Пример |
|-------|--------|
| お〜になる | お待ちになる |
| お〜する | お待ちする |
| お〜ください | お待ちください |
| お〜できる | お待ちできる |

**Когда НЕ используется:** Заимствования (サボる), гл. с одним слогом (見る→ご覧になる).
### 16.5 〜させていただく / 〜させてください
| Форма | Пример |
|-------|--------|
| させていただく | 休ませていただきます。 (Позвольте отдохнуть.) |
| させてください | 言わせてください。 (Позвольте сказать.) |
### 16.6 敬語 в вопросах
| Обычный | 敬語 |
|---------|------|
| いますか | いらっしゃいますか |
| 行きますか | おいでになりますか |
| 食べますか | 召し上がりますか |
### 16.7 Кейго: примеры в диалогах
**Диалог с начальником:**
> **A:** 部長、今お時間よろしいでしょうか。
> **B:** ええ、どうぞ。
> **A:** 先日の報告書のことですが…。
> **B:** ああ、拝見しました。とてもよくできていましたよ。
> **A:** ありがとうございます。それでは、来週の会議の資料を準備いたします。
> **B:** お願いします。何かあれば遠慮なく言ってください。

**Диалог с клиентом:**
> **A:** 田中さん、お久しぶりです。
> **B:** お久しぶりです。お元気ですか。
> **A:** はい、おかげさまで。今日はどういったご用件でしょうか。
> **B:** 新しいプロジェクトのご相談に参りました。
> **A:** そうですか。どうぞ、おかけください。お飲み物はいかがですか。
> **B:** お水をいただければ幸いです。
### 16.8 Кейго: таблица соответствий
| Обычный | 現代敬語 (совр. вежл.) | お〜になる | お〜する |
|---------|----------------------|-----------|---------|
| 行く | 行きます | いらっしゃる | 参る |
| 来る | 来ます | いらっしゃる | 参る |
| いる | います | いらっしゃる | おる |
| 言う | 言います | おっしゃる | 申す |
| 見る | 見ます | ご覧になる | 拝見する |
| 聞く | 聞きます | お聞きになる | 伺う |
| 食べる | 食べます | 召し上がる | いただく |
| 飲む | 飲みます | 召し上がる | いただく |
| する | します | なさる | 致す |
| 知る | 知っています | ご存じだ | 存じる |
| あげる | あげます | — | 差し上げる |
| もらう | もらいます | — | いただく |
| いる | います | おいでだ | おる |
| 言う | 言います | おっしゃる | 申す |
| できる | できます | なさる | できる |
### 16.9 Кейго: распространённые ошибки
| Ошибка | Правильно | Пояснение |
|--------|-----------|----------|
| おっしゃいます (свой глагол) | 言います | おっしゃる = чужой глагол |
| 召し上がります (свой глагол) | 食べます | 召し上がる = чужой глагол |
| ご覧になります (свой глагол) | 見ます | ご覧になる = чужой глагол |
| 参ります (свой глагол, разговорн.) | 行きます | 参る = скромная форма |
| 致します (свой глагол) | します | 致す = скромная форма |
| お食べになります (без необходимости) | 食べます | Если нет причины возвышать |

## 17. Пассивный и каузативный залоги — подробный разбор
### 17.4 Пассивный залог (受身形)
**Образование:**

| Класс | Форма | Пример |
|-------|-------|--------|
| I (U-глагол) | a-ряд + れる | 書く → 書かれる |
| II (Ru-глагол) | убрать -る + られる | 食べる → 食べられる |
| III (する) | される | する → される |
| III (来る) | 杯られる | 来る → 来られる |

**Типы пассивного залога:**

| Тип | Пример | Перевод |
|-----|--------|---------|
| Страдательный | 車にぶつかった | Меня сбила машина |
| Косвенный | 電車で足を踏まれた | В поезде наступили мне на ногу |
| Уважительный | 先生に褒められた | Был похвалён учителем |
| Спонтанный | 雨に降られた | Меня застал дождь |

**Примеры:**
- 犬に噛まれた。 (Меня укусила собака.)
- 友達に金を借りた。 (Друг занял у меня деньги.)
- 雨に降られた。 (Меня застал дождь.)
- 先生に叱られた。 (Меня отругал учитель.)
- 電車で財布を盗まれた。 (В поезде у меня украли кошелёк.)
### 17.5 Каузативный залог (使役形)
**Образование:**

| Класс | Форма | Пример |
|-------|-------|--------|
| I (U-глагол) | a-ряд + せる | 書く → 書かせる |
| II (Ru-глагол) | убрать -る + させる | 食べる → 食べさせる |
| III (する) | させる | する → させる |
| III (来る) | 来させる | 来る → 来させる |

**Значения:**

| Значение | Пример | Перевод |
|---------|--------|---------|
| Разрешение | 子供に遊ばせる | Позволяю ребёнку играть |
| Заставление | 子供に勉強させる | Заставляю ребёнка учиться |
| Косвенное | 先生が学生に読ませる | Учитель заставляет студента читать |

**Примеры:**
- 母は子供に野菜を食べさせる。 (Мать заставляет ребёнка есть овощи.)
- 先生は学生に日本語を話させる。 (Учитель заставляет студентов говорить по-японски.)
- 親は子供に自由に遊ばせる。 (Родители позволяют детям свободно играть.)
### 17.6 Каузативный пассив (使役受身形)
**Образование:** каузатив + passive: 書かせられる, 食べさせられる

**Значение:** Меня заставили сделать что-то (с нежеланием).

**Примеры:**
- 宿題をさせられた。 (Меня заставили сделать дз.)
- 走らされた。 (Меня заставили бегать.)
- 三時間も待たされた。 (Меня заставили ждать целых 3 часа!)
- 人に煙を吸わされた。 (Рядом курят — вынужден вдыхать дым.)
### 17.7 Пассив vs действительная форма: сравнение
| Действ. | Пассив | Разница |
|---------|--------|---------|
| 先生が学生を褒めた | 学生は先生に褒められた | Учитель похвалил студента → Студент был похвалён |
| 犬が私を噛んだ | 私は犬に噛まれた | Собака укусила меня → Меня укусила собака |
| 友達が私を助けてくれた | 私は友達に助けられた | Друг помог мне → Мне помог друг |

### 18.3 Побуждение: примеры
**Мягкое побуждение:**
- コーヒーを飲みませんか。 (Не хотите выпить кофе?)
- 映画を見に行きませんか。 (Не хотите пойти в кино?)
- 一緒に食べましょう。 (Давайте поедим вместе.)

**Настойчивое побуждение:**
- もう少し食べなさい。 (Ещё немного поешь.)
- 静かにしなさい。 (Тихо будь.)
- 遅刻しないでください。 (Не опаздывай.)

**Грубое побуждение:**
- 早くしろ！ (Быстрее!)
- 黙れ！ (Заткнись!)
- 行け！ (Иди!)

**Просьба о помощи:**
- 手伝ってください。 (Помогите, пожалуйста.)
- 少し待ってください。 (Подождите немного.)
- これを届けてください。 (Доставьте это.)

- 触らないでください。 (Не трогайте.)
- 立ってはいけない。 (Нельзя стоять.)
- 音を出してはいけない。 (Нельзя шуметь.)

**Внешний вид и кажимость:**

| Конструкция | Значение | Пример |

## 17. Подробные объяснения грамматики N4
### 17.1 て-форма с みる (попробовать)
**Формула:** て-форма + みる

**Значение:** Попробовать сделать что-то (новый опыт).

**Примеры:**

| Предложение | Перевод | Комментарий |
|-------------|---------|------------|
| 食べてみる | Попробую съесть | Новый опыт еды |
| 着てみる | Примерю (одежду) | Новый опыт примерки |
| 行ってみたい | Хочу попробовать сходить | Новый опыт поездки |
| 読んでみる | Попробую прочитать | Новый опыт чтения |
| 買ってみる | Попробую купить | Новый опыт покупки |
### 17.2 て-форма с しまう (завершённость/сожаление)
**Формула:** て-форма + しまう

**Значение:**
1. Завершённость действия (полностью сделал)
2. Сожаление (сделал, но жалею)

**Примеры:**

| Предложение | Перевод | Оттенок |
|-------------|---------|--------|
| 食べてしまった | Съел полностью | Завершённость |
| 忘れてしまった | Забыл (к сожалению) | Сожаление |
| 買ってしまった | Купил (жаль) | Сожаление |
| 読んでしまった | Прочитал (поздно) | Завершённость |

**Разговорные сокращения:**
- てしまう → ちゃう
- でしまう → じゃう

| Полная форма | Разговорная |
|-------------|-------------|
| 食べてしまう | 食べちゃう |
| 読んでしまう | 読んじゃう |
| 買ってしまう | 買っちゃう |
| 忘れてしまう | 忘れちゃう |

**Формула:** て-форма + おく

| 洗っておく | Помыть заранее | Подготовка |

**Разговорное сокращение:** ておく → とく

| Полная форма | Разговорная |
|-------------|-------------|
| 買っておく | 買っとく |
| 読んでおく | 読んどく |

### 18.1 Пассивный залог (受身形)
**Формула:**
- I класс: основа-а + れる
- II класс: убрать -る + られる
- III класс: される / 来られる

**Типы пассива:**

**Примеры:**

### 18.2 Каузативный залог (使役形)
**Формула:**
- I класс: основа-а + せる
- II класс: убрать -る + させる
- III класс: させる / 来させる

| Тип | Пример | Перевод |
