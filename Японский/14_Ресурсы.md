## 2. Бесплатные онлайн курсы

## 3. Мобильные приложения

- **Платформы:** iOS, Android

## 9. Корейские и японские ресурсы

### Различия между корейским и японским
- **Грамматика:** Похожая структура SOV
- **Система вежливости:** Аналогичные уровни формальности
- **Счётные слова:** Похожие системы
- **Письменность:** Японский использует иероглифы + кана

### Как использовать знание корейского для японского
1. Грамматические конструкции похожи
2. Некоторые слова заимствованы
3. Система вежливости аналогична
4. Порядок слов одинаковый

### Как использовать знание корейского для японского
1. Грамматические конструкции похожи
2. Некоторые слова заимствованы
3. Система вежливости аналогична

- **Письменность:** Японский использует иероглифы + кана

### Как использовать знание корейского для японского
1. Грамматические конструкции похожи
2. Некоторые слова заимствованы
3. Система вежливости аналогична
4. Порядок слов одинаковый

- **Счётные слова:** Похожие системы
- **Письменность:** Японский использует иероглифы + кана

### Как использовать знание корейского для японского
1. Грамматические конструкции похожи
2. Некоторые слова заимствованы
3. Система вежливости аналогична
4. Порядок слов одинаковый

## 126. Различия между корейским и японским

### Грамматика
- **Порядок слов:** SOV (оба языка)
- **Система вежливости:** Аналогичные уровни формальности
- **Счётные слова:** Похожие системы
- **Письменность:** Японский использует иероглифы + кана

### Как использовать знание корейского для японского
1. Грамматические конструкции похожи
2. Некоторые слова заимствованы
3. Система вежливости аналогична
4. Порядок слов одинаковый
