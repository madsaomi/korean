# Полезные ресурсы для изучения японского языка

> **Подборка лучших ресурсов для освоения языка: от YouTube-каналов и мобильных приложений до фильмов, книг и игр.**

---

## 1. YouTube каналы для изучения японского
- **JapanesePod101** — https://www.youtube.com/c/japanesepod101
  - Тысячи видео по грамматике, сленгу, аниме, путешествиям.
  - Подходит для всех уровней — от абсолютного нуля до продвинутого.
- **Learn Japanese with JapanesePod101.com** — https://www.youtube.com/user/japaneselessons101
  - Подробные объяснения грамматики и практические уроки для начинающих.
- **Misa's Japanese** — https://www.youtube.com/c/misajapanese
  - Короткие и понятные уроки по произношению, сленгу и базовой лексике.
- **Comprehensible Japanese** — https://www.youtube.com/c/ComprehensibleJapanese
  - Уроки и понятный ввод на японском языке для развития навыков восприятия на слух.
- **Nihongo con Teppei** — https://www.youtube.com/c/nihongoconteppeijapanese
  - Подкасты и разговорные видео на японском языке для тех, кто уже знает основы.

## 2. Бесплатные онлайн курсы
- **Japan Foundation Minato** — https://minato-jf.jp
  - Бесплатные курсы от Японского фонда по всем 4 навыкам (чтение, письмо, аудирование, говорение).
  - Интерактивные упражнения и доступ к учебникам Marugoto.
- **Irodori: Japanese for Life in Japan** — https://www.irodori.jpf.go.jp/en/
  - Бесплатные учебники, рабочие тетради и аудио файлы для Starter, Elementary 1 и 2.
  - Практика повседневных ситуаций и жизни в Японии.
- **NHK WORLD WEB EASY** — https://www3.nhk.or.jp/news/easy/
  - Новости на простом японском языке с фуриганой для иероглифов и аудио произношением. Отличный ресурс для уровня N4+.

## 3. Мобильные приложения
- **Duolingo** (iOS, Android)
  - Геймифицированное приложение, идеально подходит для формирования привычки, хотя не учит глубокой грамматике.
- **Anki** (Desktop, Android - бесплатно; iOS - платно)
  - Интервальные карточки (Spaced Repetition) для эффективного запоминания слов. Доступны готовые колоды с аудио.
- **WaniKani** — https://wanikani.com
  - Систематическое изучение иероглифов с использованием мнемоник. (Платная подписка с бесплатным пробным периодом).
- **Bunpro** — https://bunpro.jp
  - Практика и систематическое повторение грамматики по уровням JLPT с примерами предложений.
- **Memrise** — https://www.memrise.com
  - Интерактивные курсы с элементами геймификации для расширения словарного запаса.
- **Speechling** — https://speechling.com
  - Практика произношения и говорения с возможностью получения обратной связи от носителей языка.
- **Forvo** — https://forvo.com
  - База данных произношений слов носителями языка.

## 4. Словари и переводчики

- **Takoboto** — https://takoboto.jp

## 9. Корейские и японские ресурсы
- **Сходства языков:**
  - *Грамматика:* Практически идентичная структура предложения SOV (Субъект-Объект-Глагол).
  - *Вежливость:* Аналогичные разветвлённые уровни вежливости и формальности.
  - *Счётные слова:* Похожие принципы исчисления предметов.
- **Использование корейского при изучении японского:**
  - Грамматические конструкции очень близки, что позволяет быстрее осваивать японский синтаксис.
  - Сходный порядок слов и общие заимствования облегчают понимание лексики.

## 10. Песни для изучения японского
- **YOASOBI** (J-pop)
  - Популярная группа с быстрыми, но интересными текстами. Рекомендуемые песни: *"Idol"*, *"Gunjo"*.
- **Official HIGE DANdism** (J-pop)
  - Мелодичные песни с простой лексикой. Рекомендуемые песни: *"Pretender"*, *"Cry Baby"*.
- **King Gnu** (J-rock)
  - Яркие рок-песни с простыми фразами. Рекомендуемые песни: *"Ichizu"*, *"Hakujitsu"*.
- **Utada Hikaru** (Pop/R&B)
  - Медленные и очень выразительные песни. Рекомендуемая песня: *"First Love"*.
- **Arashi** (J-pop)
  - Весёлые песни с простой лексикой. Рекомендуемая песня: *"Love so sweet"*.
- **Традиционная музыка:**
  - *Энка (演歌)* — традиционный японский эстрадный жанр.
  - *Фолк* — японские народные песни.
  - *Классическая музыка* — произведения японских композиторов.
- **Как учить по песням:**
  1. Слушайте песню, одновременно читая её текст.
  2. Разбирайте и выписывайте новые слова.
  3. Подпевайте исполнителю для тренировки беглости произношения.
  4. Переводите тексты песен самостоятельно.

## 11. Фильмы и сериалы для обучения
- **Аниме-фильмы для начинающих:**
  - *"Your Name" (君の名は)* — романтическое аниме с повседневной лексикой.
  - *"Spirited Away" (千と千尋の神隠し)* — классика Хаяо Миядзаки с более простым сказочным языком.
  - *"Weathering with You" (天気の子)* — современная драма о погоде и жизни в Токио.
  - *"My Neighbor Totoro" (となりのトトロ)* — очень простая лексика, идеально для первого просмотра.
- **Аниме-фильмы для продвинутых:**
  - *"The Tale of the Princess Kaguya" (かぐや姫の物語)* — традиционная сказка со сложным языком.
  - *"Howl's Moving Castle" (ハウルの動く城)* — фэнтези с большим количеством специфических терминов.
- **Сериалы и шоу:**
  - *"Terrace House"* — реалити-шоу, лучший ресурс для изучения реального разговорного языка и сленга.
  - *"Midnight Diner" (深夜食堂)* — спокойная драма о жизни обычных токийцев.
  - *"Japanese Style Originator"* — документальное шоу о культуре и традициях Японии.
  - *"Alice in Borderland" (今際の国のアリス)* — триллер со средней сложностью диалогов.
  - *"The Naked Director" (全裸監督)* — историческая драма (рекомендуется для продвинутого уровня).
  - *"Giri/Haji"* — детективный триллер с двуязычными диалогами.
- **Как смотреть для обучения:**
  1. Смотрите видео с японскими (не русскими) субтитрами.
  2. Делайте паузы и повторяйте фразы вслух (метод теневого повторения).
  3. Записывайте новые выражения.
  4. Просматривайте сложные сцены по нескольку раз.

## 12. Книги и учебники
- **Учебные пособия:**
  - *"Genki"* — один из самых популярных и понятных англо-японских учебников для начинающих.
  - *"Minna no Nihongo"* — классический учебник, полностью на японском (с отдельными грамматическими книгами-переводами).
  - *"Japanese Made Easy"* — современный самоучитель с акцентом на бытовую речь.
  - *"Tobira"* — отличный мост от начального уровня к среднему (Intermediate).
- **Манга для начинающих (простой язык):**
  - *"Yotsuba&!" (よつばと!)* — повседневная жизнь девочки, минимум сложных кандзи.
  - *"Doraemon" (ドラえもん)* — классические детские сюжеты.
  - *"Azumanga Daioh" (あずまんが大王)* — школьная комедия.
- **Художественные книги:**
  - *Книги с параллельным текстом* — истории с построчным переводом.
  - *"Short Stories in Japanese"* — адаптированные короткие рассказы.
  - *"Japanese Folk Tales"* — японские народные сказки в простой обработке.
  - *"Harry Potter" на японском* — знакомый сюжет облегчает понимание сложных слов.
  - *"Norwegian Wood" (ノルウェイの森)* — роман Харуки Мураками (для продвинутых).
