## 2. Введение в корейский язык

### 2.1 Общая информация

- **Язык:** 한국어 (хангыго)
- **Число носителей:** ~77 миллионов
- **Семья:** Корейская (изолированная)
- **Тип письма:** Слоговое (алфавит Хангыль)
- **Порядок слов:** SOV (подлежащее-объект-сказуемое)
- **Грамматика:** Агглютинативная, темо-рематическая

### 2.2 Система письма

| Система | Тип | Назначение |
|---------|-----|-----------|
| **한글 (хангыль)** | Алфавит | Основная система письма |
| **한자 (ханджа)** | Китайские иероглифы | Используются ограниченно |

### 2.3 Уровни владения

| Уровень | TOPIK | Часы | Что можете |
|---------|-------|------|-----------|
| Начальный | TOPIK I | ~150 | Базовые фразы, простые диалоги |
| Средний | TOPIK II | ~300 | Повседневные темы |
| Продвинутый | TOPIK III-IV | ~600 | Сложные тексты |
| Высокий | TOPIK V-VI | ~1000 | Свободное общение |

### 2.4 Особенности корейского языка

1. **Алфавит Хангыль** — один из самых научных алфавитов в мире
2. **Нет артиклей** — нет аналогов "a/the"
3. **Нет рода** — нет деления на мужской/женский
4. **Вежливость** — встроена в грамматику (5 уровней)
5. **Слоговое письмо** — каждый символ = один слог
6. **Порядок слов** — SOV (как в японском)
7. **Заимствования** — много слов из китайского и английского

### 2.5 Интересные факты о корейском

1. Хангыль был изобретён в 1443 году королём Седжоном Великим
2. Хангыль считается одним из самых научных алфавитов
3. В корейском 14 согласных и 10 гласных
4. Каждый символ Хангыль = 1 слог
5. Корейский очень логичен в своей структуре
6. Нет ударения (в отличие от японского)
7. Много заимствований из китайского (70% лексики)
8. Много заимствований из английского (современные термины)

### 2.6 Структура слога

Корейский символ состоит из 2-4 элементов:

| Тип | Структура | Пример |
|-----|-----------|--------|
| Открытый слог | Согласный + гласный | 가 = ㄱ + ㅏ |
| Закрытый слог | Согласный + гласный + придача | 감 = ㄱ + ㅏ + ㅁ |
| Открытый (2 элемента) | Согласный + гласный | 가 = ㄱ + ㅏ |
| Закрытый (3 элемента) | Согласный + гласный + придача | 감 = ㄱ + ㅏ + ㅁ |

## 3. Система вежливости

Корейский язык имеет развитую систему вежливых форм:
- Неформальная: -아/어
- Вежливая: -요
- Формальная: -습니다/ㅂ니다

---

### 3.1 Уровни вежливости

| Уровень | Форма | Когда использовать |
|---------|-------|-------------------|
| Формальная | -ㅂ니다 | С незнакомцами, на работе |
| Нейтральная | -아/어요 | С друзьями, знакомыми |
| Разговорная | -아/어 | С близкими друзьями |
| Грубая | базовая | Не используется |

## 17. Расширенное введение: Корейский язык

### 17.1 Место корейского языка

Корейский язык — официальный язык Южной Кореи (Республика Корея) и Северной Кореи (Корейская Народно-Демократическая Республика). На нём говорят около 80 миллионов человек по всему миру.

### 17.2 Связи с другими языками

Корейский язык имеет некоторые связи с японским языком:
- Общая грамматическая структура (SOV)
- Похожая система вежливых форм
- Аналогичная система счётных слов

---

### 17.3 Структура алфавита

Корейский алфавит (한글/Hangul) состоит из:
- 14 согласных: ㄱ ㄴ ㄷ ㄹ ㅁ ㅂ ㅅ ㅇ ㅈ ㅊ ㅋ ㅌ ㅍ ㅎ
- 10 гласных: ㅏ ㅓ ㅗ ㅜ ㅡ ㅣ ㅐ ㅔ ㅚ ㅟ
- 5 двойных согласных: ㄲ ㄸ ㅃ ㅆ ㅉ

### 17.4 Система слогов

Корейский алфавит организован в слоги:
- Конsonant + Vowel: 가 (ка)
- Конsonant + Vowel + Конsonant: 감 (кам)
- Конsonant + Vowel + Конsonant + Конsonant: 앉 (андж)

---

Корейский алфавит организован в слоги:
- Консонant + Vowel: 가 (ка)
- Консонant + Vowel + Консонant: 감 (кам)
- Консонant + Vowel + Консонant + Консонant: 앉 (андж)

### 17.5 Базовая структура предложения

Корейский язык использует порядок слов SOV (Подлежащое + Дополнение + Сказуемое):
- Я читаю книгу: 저는 책을 읽어요.
- Он пьёт воду: 그는 물을 마셔요.
- Мы едим рис: 우리는 밥을 먹어요.

### 17.6 Основные правила

1. Нейтрализация: ㅂ → [п], ㄷ → [т], ㄱ → [к]
2. Назализация: ㄴ + ㄹ → [лл]
3. Смягчение: ㄷ + ㅣ → [дж]
4. Удвоение: ㄲ → [кк], ㅃ → [пп]

### 17.7 Сложные случаи

1. Промежуточное ㄹ: 달라 → [далла]
2. Начальное ㅇ: 아 → [а]
3. Конечное ㅇ: 감 → [кам]

## 18. Расширенное введение: Письменность

### 18.1 Типы письма

1. 한글 (Хангыль) — основной алфавит
2. 한자 (Ханча) — китайские иероглифы (редко)
3. 로마자 (Ромадза) — латиница (для транслитерации)

### 18.2 Правила написания

1. Слоги пишутся слева направо и сверху вниз
2. Согласные и гласные чередуются
3. Конечные согласные (받침) обязательны

---
### 18.3 Структура слогов

Корейский язык пишется «блоками», каждый из которых содержит один слог. Блоки ВСЕГДА рисуются одним из следующих способов:

**Правила:**
1. Позиция 2 — ВСЕГДА гласная
2. Позиции 1 и 3 (иногда 4) — ВСЕГДА согласные
3. Блоки с горизонтальной гласной: ㅂ + ㅏ + ㅂ = 밥 (рис)
4. Блоки с вертикальной гласной: ㄱ + ㅏ + ㄴ = 간 (печень)
