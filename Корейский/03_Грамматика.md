## 1. Частицы (조사)

### 1.1 Основные частицы

| Частица | Функция | Пример |
|---------|---------|--------|
| 은/는 | Тема | 저는 학생입니다 |
| 이/가 | Субъект | 책이 있습니다 |
| 을/를 | Объект | 책을 읽습니다 |
| 에 | Направление/время | 학교에 갑니다 |
| 에서 | Место действия | 학교에서 공부합니다 |
| 과/와 | Совместность | 친구와 갑니다 |
| 의 | Принадлежность | 나의 책 |
| 도 | Тоже | 나도 갑니다 |
| 부터 | От (время) | 아침부터 |
| 까지 | До (время/место) | 저녁까지 |
| 만 | Только | 나만 갑니다 |
| 조차 | Даже | 이름조차 모릅니다 |
| 마저 | Даже | 비까지 왔습니다 |
| 밖에 | Только (отриц.) | 이것밖에 없습니다 |
| 뿐 | Только | 이것뿐입니다 |

### 1.2 Различия 은/는 и 이/가

| Ситуация | 은/는 | 이/가 |
|----------|-------|-------|
| Тема (уже известная) | 나는 학생입니다 | — |
| Субъект (новый) | — | 책이 있습니다 |
| Контраст | 나는 먹지만 | — |
| Вопрос с «кто» | — | 누가 왔습니까? |
| Ограничение | — | 나만 |

### 1.3 Различия 에 и 에서

| Ситуация | 에 | 에서 |
|----------|----|------|
| Направление | 학교에 갑니다 | — |
| Место (существование) | 집에 있습니다 | — |
| Место (действие) | — | 학교에서 공부합니다 |
| Время | 아홉시에 | — |
| Средство | — | 버스로 갑니다 |

|-----|-----------|--------|

| Правильные | -다 | 읽다, 앉다, 일어나다 |

| Глагол | Значение | ㅂ니다 | н/는다 |过去 |

| 공부하다 | Учиться | 공부합니다 | 공부한다 | 공부했다 |
| 읽다 | Читать | 읽습니다 | 읽는다 | 읽었다 |

## 3. Прилагательные

### 3.1 Типы прилагательных

| Тип | Окончание | Пример |
|-----|-----------|--------|
| ㅏ/ㅗ | -ㅂ니다 | 좋습니다, 작습니다 |
| 그/으 | -ㅂ니다 | 크습니다, 예쁩니다 |
| ㅎ | -ㅂ니다 | 높습니다 |

### 3.2 Основные прилагательные

| Прилагательное | Значение | ㅂ니다 |比较 |
|----------------|---------|--------|-----|
| 좋다 | Хороший | 좋습니다 | 좋다/좋지 않다 |
| 나쁘다 | Плохой | 나쁩니다 | 나쁘다/나쁘지 않다 |
| 크다 | Большой | 큽니다 | 크다/작지 않다 |
| 작다 | Маленький | 작습니다 | 작다/크지 않다 |
| 비싸다 | Дорогой | 비쌉니다 | 비싸다/싸지 않다 |
| 싸다 | Дешёвый | 싸습니다 | 싸다/비싸지 않다 |
| 맛있다 | Вкусный | 맛있습니다 | 맛있다/맛없다 |
| 예쁘다 | Красивый | 예쁩니다 | 예쁘다/예쁘지 않다 |
| 어렵다 | Сложный | 어렵습니다 | 어렵다/ 쉽지 않다 |
| 쉽다 | Лёгкий | 쉽습니다 | 쉽다/어렵지 않다 |
| 높다 | Высокий | 높습니다 | 높다/낮지 않다 |
| 낮다 | Низкий | 낮습니다 | 낮다/높지 않다 |
| 빠르다 | Быстрый | 빠릅니다 | 빠르다/느리지 않다 |
| 느리다 | Медленный | 느립니다 | 느리다/빠르지 않다 |

### 3.3 Степени сравнения

| Конструкция | Пример | Перевод |
|-------------|--------|---------|
| A가 B보다 ~ | 한국어가 일본어보다 쉽습니다 | Корейский легче японского |
| A보다 B가 ~ | 일본어보다 한국어가 쉽нее | Корейский легче |
| 가장/제일 ~ | 가장 쉬운 언어 | Самый лёгкий язык |
| ~처럼 | 한국어처럼 쉽다 | Лёгкий как корейский |

## 7. Условные конструкции

### 7.1 Все условные конструкции

| Конструкция | Формула | Значение | Пример |
|-------------|---------|---------|--------|
| -면 | Глагол + 면 | Если | 가면 |
| -아/어도 | Глагол + 아/어도 | Даже если | 가도 |
| -ㄹ 때 | Глагол + ㄹ 때 | Когда | 갈 때 |
| -ㄹ 테니 | Глагол + ㄹ 테니 | Поскольку | 갈 테니 |
| -아/어 봐야지 | Глагол + 아/어 봐야지 | Обязательно попробую | 가 봐야지 |

### 7.2 Подробные примеры

**-면 (если):**
| Пример | Перевод | Оттенок |
|--------|---------|--------|
| 가면 | Если пойду | Условие |
| 먹으면 | Если поем | Условие |
| 비싸면 | Если дорого | Условие |

**-아/어도 (даже если):**
| Пример | Перевод | Оттенок |
|--------|---------|--------|
| 가도 | Даже если пойду | Уступка |
| 먹어도 | Даже если поем | Уступка |
| 비싸도 | Даже если дорого | Уступка |

**-ㄹ 때 (когда):**
| Пример | Перевод | Оттенок |
|--------|---------|--------|
| 갈 때 | Когда пойду | Момент |
| 먹을 때 | Когда поем | Момент |
| 잘 때 | Когда сплю | Момент |

### 120.1 Глагол существования (있다/없다)

**Базовые примеры**
- 책상 위에 책이 있어요. — На столе есть книга.
- 냉장고에 음식이 없어요. — В холодильнике нет еды.
- 여기에 화장실이 있어요? — Здесь есть туалет?
- 저기에 공원이 있어요. — Там есть парк.

**Сложные примеры**
- 한국에 친구가 있어요. — В Корее есть друг.
- 저는 여동생이 있어요. — У меня есть младшая сестра.
- 시간이 없어요. — Нет времени.
- 돈이 없어요. — Нет денег.
- 걱정이 있어요. — Есть беспокойство.
- 문제가 있어요. — Есть проблема.

**Отрицательные примеры**
- 저는 여동생이 없어요. — У меня нет младшей сестры.
- 여기에 펜이 없어요. — Здесь нет ручки.
- 시간이 없어요. — Нет времени.
- 돈이 없어요. — Нет денег.
- 걱정이 없어요. — Нет беспокойства.
- 문제가 없어요. — Нет проблемы.

- 책상 위에 책이 있어요. — На столе есть книга.
- 냉장고에 음식이 없어요. — В холодильнике нет еды.
- 여기에 화장실이 있어요? — Здесь есть туалет?
- 저기에 공원이 있어요. — Там есть парк.

**Сложные примеры**
- 한국에 친구가 있어요. — В Корее есть друг.
- 저는 여동생이 있어요. — У меня есть младшая сестра.
- 시간이 없어요. — Нет времени.
- 돈이 없어요. — Нет денег.
- 걱정이 있어요. — Есть беспокойство.
- 문제가 있어요. — Есть проблема.

**Отрицательные примеры**
- 저는 여동생이 없어요. — У меня нет младшей сестры.
- 여기에 펜이 없어요. — Здесь нет ручки.
- 시간이 없어요. — Нет времени.
- 돈이 없어요. — Нет денег.
- 걱정이 없어요. — Нет беспокойства.
- 문제가 없어요. — Нет проблемы.

### 121.2 Глагол желания (고 싶다)

**Базовые примеры**
- 물을 마시고 싶어요. — Хочу пить воду.
- 한국에 가고 싶어요. — Хочу поехать в Корею.
- 영화를 보고 싶어요. — Хочу посмотреть фильм.
- 음식을 먹고 싶어요. — Хочу поесть.

**Сложные примеры**
- 저는 한국어를 배우고 싶어요. — Я хочу выучить корейский.
- 우리는 여행을 가고 싶어요. — Мы хотим путешествовать.
- 저는 운동을 하고 싶어요. — Я хочу заниматься спортом.
- 저는 독서를 하고 싶어요. — Я хочу читать.
- 저는 영화를 보고 싶어요. — Я хочу посмотреть фильм.
- 저는 음악을 듣고 싶어요. — Я хочу слушать музыку.

**С негацией**
- 저는 운동을 하기 싫어요. — Я не хочу заниматься спортом.
- 저는 일하기 싫어요. — Я не хочу работать.
- 저는 공부하기 싫어요. — Я не хочу учиться.

### 121.3 Прошедшая форма (-었어요/았어요)

- 저는 어제 책을 읽었어요. — Вчера я читал книгу.

**Сложные примеры**

- 저는 지난 달에 돈을 썼어요. — В прошлом месяце я потратил деньги.
- 저는 어제 시험을 봤어요. — Вчера я сдавал экзамен.
- 저는 어제 친구를 만났어요. — Вчера я встречался с другом.
- 저는 어제 숙제를 했어요. — Вчера я делал домашнее задание.

**С негацией**
- 저는 어제 영화를 안 봤어요. — Вчера я не смотрел фильм.
- 저는 어제 친구를 안 만났어요. — Вчера я не встречался с другом.
- 저는 어제 음식을 안 먹었어요. — Вчера я не ел еду.

## 125. Грамматика: Негативные конструкции

### 125.1 Отрицание с 안
- 저는 영화를 안 봐요. — Я не смотрю фильм.
- 저는 음악을 안 들어요. — Я не слушаю музыку.
- 저는 책을 안 읽어요. — Я не читаю книгу.
- 저는 운동을 안 해요. — Я не занимаюсь спортом.

### 125.2 Отрицание с 못
- 저는 한국어를 못 해요. — Я не умею говорить по-корейски.
- 저는 수영을 못 해요. — Я не умею плавать.
- 저는 요리를 못 해요. — Я не умею готовить.
- 저는 노래를 못 해요. — Я не умею петь.

### 125.3 Разница между 안 и 못
- 안 = «не делаю» (сознательно)
- 못 = «не могу сделать» (объективно)
- 저는 영화를 안 봐요. — Я не смотрю фильм (не хочу).
- 저는 영화를 못 봐요. — Я не могу посмотреть фильм (нет времени/возможности).

## 166. Грамматика: Степени сравнения

## 167. Грамматика: Условные конструкции

### 167.1 Если... то... (-면/으면)

- 비가 오면 우산을 가져가요. — Если идёт дождь, беру зонтик.

### 167.2 Если не... то... (-지 않으면)

- 비가 오지 않으면 우산을 안 가져가요. — Если не идёт дождь, зонт не беру.

## 168. Грамматика: Причинные конструкции

### 168.2 Из-за того, что (-땜에/때문에)

- 교통 때문에 늦었어요. — Из-за транспорта опоздал.
