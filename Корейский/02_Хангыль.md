## 4. Гласные (10 базовых)

### 4.1 Простые гласные

| Символ | Чтение | МФА | Описание | Пример |
|--------|--------|-----|----------|--------|
| ㅏ | a | [a] | Как «а» в «мама» | 아 (a) |
| ㅑ | ja | [ja] | Как «я» в «яма» | 야 (ya) |
| ㅓ | ŏ | [ʌ] | Между «о» и «а» | 어 (eo) |
| ㅕ | jŏ | [jʌ] | Как «ё» (открытый) | 여 (yeo) |
| ㅗ | o | [o] | Как «о» в «окно» | 오 (o) |
| ㅛ | jo | [jo] | Как «ё» в «ёлка» | 요 (yo) |
| ㅜ | u | [u] | Как «у» в «лук» | 우 (u) |
| ㅠ | ju | [ju] | Как «ю» в «юла» | 유 (yu) |
| ㅡ | ŭ | [ɯ] | Безударное «ы» | 으 (eu) |
| ㅣ | i | [i] | Как «и» в «мир» | 이 (i) |

### 4.2 Типы слогов

| Тип | Структура | Пример | Описание |
|-----|-----------|--------|----------|
| Открытый | ㄱ + ㅏ | 가 (ga) | Согласный + гласный |
| Закрытый | ㄱ + ㅏ + ㅁ | 감 (gam) | Согласный + гласный + придача |

### 4.3 Как записывать слоги

**Открытый слог:**
```
ㄱ + ㅏ = 가
```

**Закрытый слог:**
```
ㄱ + ㅏ + ㅁ = 감
```

### 19.3 Секреты чтения слогов

**Лайфхак 1: Формула слога**

```
[Согласная] + [Гласная] = Слог
ㄱ + ㅏ = 가 (ga)
ㄴ + ㅓ = 너 (neo)
ㄷ + ㅗ = 도 (do)
```

**Лайфхак 2: Структура слога**

```
Прямой: Согласная + Гласная
  ㄱ + ㅏ = 가

Обратный: Гласная + Согласная
  ㅏ + ㄱ = 각

Комбинированный: Согласная + Гласная + Придача
  ㄱ + ㅏ + ㅁ = 감
```

**Лайфхак 3: Правила чтения**

| Правило | Пример |
|---------|--------|
| ㄱ/ㅋ/ㄲ → [k] | 약 [як] |
| ㄴ → [n] | 산 [сан] |
| ㄷ/ㅌ/ㅆ → [t] | 옷 [от] |
| ㄹ → [l] | 달 [тал] |
| ㅁ → [m] | 밤 [пам] |
| ㅂ/ㅍ → [p] | 밥 [пап] |
| ㅅ/ㅆ/ㅈ/ㅊ/ㅎ → [t] | 옷 [от] |
| ㅇ → [ng] | 방 [бан] |

### 19.4 Секреты озвончения

**Лайфхак: Озвучивайте после гласных и ㄴ, ㅁ**

| Придача | После гласной/ㄴ, ẩm | После других |
|---------|---------------------|-------------|
| ㄱ | ㅇ | ㄱ |
| ㄷ | ㄴ | ㄷ |
| ㅂ | ẩm | ㅂ |
| ㅅ | ㅇ | ㅅ |
| ㅈ | ㅇ | ㅈ |

**Примеры:**
- 약 [як] → 약 [як] (после ㄱ)
- 산 [сан] → 산 [сан] (после ㄴ)
- 밤 [пам] → 밤 [пам] (после ẩm)
- 옷 [от] → 옷 [от] (после ㅅ)

## 22. Расширенные примеры: Произношение

### 22.1 Правила произношения

**Правило 1: Нейтрализация**
- ㅂ → [п] перед гласными: 아름답다 → [арымтапда]
- ㄷ → [т] перед гласными: 옳다 → [ольтда]
- ㄱ → [к] перед гласными: 약하다 → [яхада]

**Правило 2: Назализация**
- ㄴ + ㄹ → [лл]: 한라산 → [халласан]
- ㄹ + ㄴ → [нн]: 설날 → [сольнал]

**Правило 3: Смягчение**
- ㄷ + ㅣ → [дж]: 믿다 → [миддза]
- ㅌ + ㅣ → [ч]: 밭이 → [паджи]

**Правило 4: Удвоение**
- ㄲ → [кк]: 오빠 → [обба]
- ㅃ → [пп]: 아빠 → [аппа]
- ㅆ → [сс]: 낫다 → [натда]
- ㅉ → [дждж]: 찌개 → [джджигэ]

## 27. Произношение корейских букв

### 27.1 Важное замечание о произношении

Люди постоянно спрашивают о произношении корейских букв и как их лучше представить с помощью английского. Не существует идеального способа представить корейские буквы с помощью английских. Английские буквы выше — это те, которые обычно используются для представления соответствующих корейских букв. Хотя полезно (в начале) запомнить общий звук корейской буквы, используя английскую — нужно помнить, что корейские звуки сильно отличаются от английских.

Например, часто можно увидеть:
- «K» и «G» для ㄱ
- «D» и «T» для ㄷ
- «R» и «L» для ㄹ

Правда в том, что ни одна из этих букв не идеально соответствует звуку соответствующей корейской буквы. Единственный способ узнать точный звук корейской буквы — послушать её.

### 27.2 Практика написания

Попробуйте написать следующие слоги:

1. ㄱ + ㅏ = 가
2. ㄴ + ㅓ = 너
3. ㄷ + ㅗ = 도
4. ㄹ + ㅜ = 루
5. ㅁ + ㅣ = 미
6. ㅂ + ㅏ = 바
7. ㅅ + ㅓ = 서
8. ㅇ + ㅗ = 오
9. ㅈ + ㅜ = 주
10. ㅎ + ㅏ = 하
